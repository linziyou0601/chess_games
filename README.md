# reversi

簡單的棋類對戰伺服器

- 使用NuxtJS作為框架
- 使用Socket.io進行連線管理
- 使用mariadb儲存帳號及對戰資料

# 說明

- 請至 db.config.js 更改DB連線設定
- 請至 .env 更改BASE_URL設定
